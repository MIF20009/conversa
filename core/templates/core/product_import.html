Excel File Format Instructions
Required Columns:
  • name - Product name (required)
  • price_usd - Price in USD (required)
Optional Columns:
  • sku - Product SKU code
  • description - Product description
  • price_lbp - Price in Lebanese Pounds
  • stock - Stock quantity (defaults to 0)
Example Excel Format:
name price_usd sku stock description
T-Shirt 25.99 TSH001 50 Cotton T-Shirt
Jeans 45.00 Blue Denim
Hat 15.50 HAT123 25
Tips:
  • Use .xlsx format (Excel 2007+)
  • Leave optional columns empty if you don't have that data
  • Make sure column headers match exactly (case-sensitive)
  • Empty rows will be skipped automatically
  • Invalid rows will show error messages after upload
